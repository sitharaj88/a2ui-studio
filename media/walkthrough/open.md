# One studio, five views

**A2UI Studio** puts everything in a single panel:

- **Dashboard** — session stats, message-stream anatomy, provider status, activity feed
- **Playground** — chat with the agent, watch surfaces render live, inspect every message
- **Arena** — run one prompt against two providers side by side
- **Gallery** — instant demos and curated prompts
- **Settings** — profiles, generation tuning, connections, appearance

The status bar always shows which provider or profile is active — click it any time to jump back in.
